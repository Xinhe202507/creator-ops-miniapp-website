# 达人分销管理小程序部署说明

这个文件夹是一个完整的静态网页/PWA 小程序，不需要数据库和后端服务。

它可以直接作为网站部署：上传整个文件夹到 Vercel、Netlify、GitHub Pages、Cloudflare Pages 或公司服务器后，就会得到一个电脑和手机都能打开的 `https://...` 网站链接。

## 文件清单

- `index.html`：主页面
- `styles.css`：样式
- `app.js`：业务逻辑、双语切换、新增达人、本地保存
- `progress-utils.js`：按日期查看当天工作进度的数据工具
- `manifest.webmanifest`：手机安装/PWA 配置
- `sw.js`：离线缓存
- `icon.svg`：应用图标
- `start-local-server.bat`：Windows 电脑一键启动本地网站服务
- `vercel.json`：Vercel 网站部署配置
- `netlify.toml`：Netlify 网站部署配置
- `.nojekyll`：GitHub Pages 静态网站配置

## 电脑和手机打开方式

这个版本已经按电脑和手机浏览器做了响应式适配：

- 电脑端：左侧模块导航 + 右侧数据工作台
- 手机端：导航横向滑动，内容单列显示，表格可横向滑动，弹窗接近全屏显示

## 电脑最简单打开方式

把整个文件夹复制到其他电脑后，直接双击 `index.html` 可以打开。

注意：直接双击打开时，核心功能可以用，但手机安装到桌面和离线缓存需要通过 `http://` 或 `https://` 访问。

## Windows 电脑推荐打开方式

把整个文件夹复制到其他 Windows 电脑后，双击：

```text
start-local-server.bat
```

脚本会自动打开：

```text
http://localhost:8080/
```

这种方式比直接双击 `index.html` 更接近正式网站访问方式，也支持 PWA 缓存能力。

## 让同一局域网手机打开

1. 在这个文件夹运行静态服务。

   ```powershell
   python -m http.server 8080
   ```

2. 查电脑的局域网 IP。

   ```powershell
   ipconfig
   ```

3. 手机和电脑连接同一个 Wi-Fi，在手机浏览器打开：

   ```text
   http://电脑IP:8080/
   ```

例如：

```text
http://192.168.1.23:8080/
```

## 直接发布成网站

把整个文件夹上传到任意静态网站托管平台即可。发布后，平台会生成一个 `https://...` 链接，电脑和手机都访问同一个网址。

- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages
- 公司自己的服务器或对象存储

上传后访问平台给你的 `https://...` 链接，电脑和手机都能打开。

推荐最简单方式：

1. 打开 Netlify 或 Vercel
2. 新建项目
3. 上传或拖入这个文件夹
4. 等待部署完成
5. 复制平台生成的网址发给团队使用

## Supabase 云端同步配置

当前版本已支持 Supabase 账号密码登录和云端同步。未配置 Supabase 时，网站仍可作为本地演示版本使用；配置后，不同电脑和手机登录同一套账号体系即可看到同一份数据。

配置步骤：

1. 在 Supabase 新建项目。
2. 打开 Supabase SQL Editor，执行 `supabase-schema.sql`。
3. 在 Supabase Authentication 里创建用户账号。
4. 在 `creator_ops_users` 表里给用户配置角色：

   - `admin`：管理员，可新增、修改、删除全部数据
   - `operator`：运营，只能操作自己负责人的数据
   - `readonly`：只读，只能查看，不能修改删除

5. 打开 `supabase-config.js`，填写项目参数：

   ```js
   window.CREATOR_OPS_SUPABASE = {
     SUPABASE_URL: "https://你的项目.supabase.co",
     SUPABASE_ANON_KEY: "你的 anon key"
   };
   ```

6. 重新上传网站文件到 GitHub Pages / Vercel / Netlify。

同步范围：

- 达人 CRM 档案
- SKU
- 样品单
- 视频记录
- 直播记录
- 删除记录
- 负责人 / SKU / 样品成本记忆选项

数据同步方式：登录后网站会先从 Supabase 拉取云端数据覆盖本地缓存；后续新增、修改、删除会自动写回 Supabase。只读账号会隐藏或拦截写入操作。

## 数据保存说明

当前版本把新增达人数据保存在当前浏览器的本地存储里。

达人 CRM 表格右侧有“修改 / 删除”操作：

- 修改：打开达人表单，保存后更新该达人数据
- 删除：只删除当前点击的那一条记录；同名达人不会被一起删除
- 示例达人被修改后会以本地覆盖记录保存
- 示例达人被删除后会记录在本地隐藏列表中
- 如果删除后再次录入同名达人，系统会自动取消隐藏并重新添加到列表

样品管理页面中的“新增样品单”已经改为可点击板块，点击后会触发样品单入口提示。

新增或修改达人时，合作信息里的 SKU 会自动记录到 `商品 SKU 分销` 模块。样品单价、数量、总成本会自动记录到 `样品管理` 模块；如果把“是否需要样品”改为“否”，对应的自定义样品记录会被移除。

样品单价、样品总成本、报价/固定费用支持小数，例如 `12.50`。

`商品 SKU 分销` 模块支持直接新增和修改 SKU。点击“新增 SKU”会打开 SKU 表单；点击表格行里的“修改”可以编辑售价、成本、佣金、库存、样品库存、爆品潜力和适合达人。

`商品 SKU 分销` 模块也支持单条删除 SKU。删除只影响当前点击的 SKU，不会清空全部 SKU；如果后续重新新增同名 SKU，会重新显示。

达人合作信息里已增加“发货时间”和“收到货时间”。如果达人收到货后超过 3 天仍没有发布视频，首页风险任务和样品管理风险列会自动显示提醒。

`样品管理` 模块里的“新增样品单”已改为可输入表单，可以单独录入达人、SKU、样品单价、数量、总成本、运费、发货时间、收到货时间、发布截止和负责人。样品表格每条记录后面支持“修改 / 删除”，删除只删除当前样品单。

达人 CRM 档案支持“手动上传表格”。当前离线版本支持 Excel 另存为 `CSV` 或 `TSV` 后上传，并自动识别达人昵称、平台、负责人、合作 SKU、样品价格数量、发货时间、收到货时间、视频数据等字段。可参考 `达人导入模板.csv`。

达人 CRM 档案里的“负责人”支持手动填写；填写过的负责人会自动进入下次录入时的可选建议。合作信息里的 SKU、样品总成本也会自动记忆，后续新增达人、SKU 或视频记录时可以直接选择历史填写内容。

达人 CRM 表格已支持“查看视频”。一个达人可以在视频内容记录弹窗中添加多条视频，每条视频都可以单独修改或删除；内容发布管理模块会显示全部视频记录；ROI 分析支持按达人、按视频、按 SKU 三种视角切换。

达人 CRM 表格已支持“查看直播”。一个达人可以在直播合作记录弹窗中添加多场直播；每场直播可记录平台、日期、开始/结束时间、直播间链接、合作 SKU、样品成本、坑位费、佣金、观看人数、峰值在线、点击、订单、GMV、退款、ROI、状态、负责人和备注。每条直播记录都支持单独修改和删除。

内容发布管理模块支持“视频 / 直播”切换；ROI 分析支持按达人、按视频、按直播、按 SKU 查看。首页会提醒：直播日期已过但 GMV 或订单数未填写的直播，需要补齐复盘。

右上角日期选择器可以切换查看当天工作进度。当前示例数据包含：

- `2026-06-13`
- `2026-06-14`
- `2026-06-15`

这意味着：

- 同一台设备、同一浏览器刷新后数据不丢失
- 换电脑或换手机不会自动同步
- 如果需要团队多人共用同一份数据，下一步要接入后端数据库或 Google Sheets / Airtable / Supabase
